Data Usage Guidelines

This raw data is provided to illustrate the original state of student data and offer an opportunity for independent data cleaning exercises. Please note that utilizing this specific dataset is neither a mandatory requirement nor a guarantee of a favorable grade. Students are encouraged to leverage any data source they deem most conducive to achieving academic success within the context of the assignment.

Data Redistribution Policy

This data is explicitly NOT intended for redistribution. Any form of sharing, dissemination, or usage beyond the scope of individual student work for this assignment requires prior written consent.
